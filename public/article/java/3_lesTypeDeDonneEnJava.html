Nombres entiers :

En java il existe deux méthodes pour écrire des nombres. La première consiste à coller chaque chiffre au chiffre suivant comme par exemple 100000000. Mais cette écriture ne facilite pas la lecture pour les nombres trop grands. C’est pour cela qu’il est possible d’utiliser le séparateur « _ » pour faciliter la lecture, 100_000_000.

Manipuler les bases :

  • Manipuler des nombres binaires : rajouter 0b devant, comme ici : 0b100011110.
  • Manipuler un nombre de base octale : rajouter 0 devant le nombre.
  • Manipuler des nombres de base hexadécimale : rajouter 0x devant le nombre et écrire les lettres en majuscule.

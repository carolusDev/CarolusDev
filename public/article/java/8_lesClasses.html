Les classes, N°8

Qu’est-ce qu’une classe en Java ?

Classe : entité

Représentant un ensemble d’objets ayant la même représentation mémoire (structure) et les mêmes comportements :

  • Définissant cette structure
  • Définissant et détenant ces comportements
  • Capable de générer des instances

Composition d’une classe :

Public : représente le niveau d’accès d’une classe, portée : public (accès partout sans restriction), private (n’est pas accessible).

Nommer la classe, il faut impérativement que le nom du fichier soit le même que le nom de la classe contenue.

Ajouter entre les accolades le contenu de la classe. Si c’est la classe principale du programme (première exécutée), celle-ci doit contenir la méthode Main.

Compiler et exécuter plusieurs programmes Java :

Javac *.java permet de compiler tous les fichiers ayants comme extension .java, tandis que java App, permet d’exécuter le programme principal.
